# Ayushi Aggarwal Portfolio

A clean, minimal portfolio website for Ayushi Aggarwal - AI product builder and founder.

## Tech Stack

- **React.js** with Vite
- **Tailwind CSS** for styling
- **Framer Motion** for subtle animations
- **Lucide React** for icons

## Features

- Dark mode by default with violet accent color
- Fully responsive design
- Smooth scrolling and animations
- Project showcase with live links
- Minimal, founder-focused design

## Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

4. **Preview production build:**
   ```bash
   npm run preview
   ```

## Project Structure


## Design Notes

- **Color Scheme:** Dark background (#0A0A0A) with violet accent (#7C3AED)
- **Typography:** DM Sans from Google Fonts
- **Animations:** Subtle fade-up animations with Framer Motion
- **Responsive:** Mobile-first design with breakpoints at sm, md, lg
- **Performance:** Optimized with Vite's build tools

## Sections

1. **Hero:** Full viewport with name, tagline, and CTAs
2. **About:** Brief founder bio
3. **Projects:** Grid of 5 AI products with live links
4. **Currently:** Current focus statement
5. **Footer:** Contact and social links

The portfolio signals "builder/founder" energy through clean typography, minimal design, and focus on shipped products rather than skills or services.
